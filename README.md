# Chrome-Extension-Main-Kernels
This is the original chrome extension kernel pack including manifest.json and the popup.html or popup.md. You can add in an image too as the icon.

- How To Use
-   Click the code button on the github repo.
-   Click Download .zip
  - Extract the .zip file.
-  Open the Chrome Extension main in VS Code.
-   Edit the html of the Chrome extension ( dont't change any names. )
-   Add a .png file to the folder called logo.png. You can make it anything you want.
-   Then watch a tutorial on how to upload a unpacked folder to Chrome.
